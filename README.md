# Vividly - Weekly Calendar UI

This is a code bundle for a Weekly Calendar UI application. 

## Running the code

1. Run `npm i` to install the dependencies.
2. Run `npm run dev` to start the development server.

## Project Origin
The original design inspiration is available at [Figma](https://www.figma.com/design/veX5Wgh8OYm8ZYbqk6MNVv/Weekly-Calendar-UI).
